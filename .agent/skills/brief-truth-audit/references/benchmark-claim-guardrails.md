# Benchmark Claim Guardrails

## NEVER Claim

- "NotebookLM parity" or "NotebookLM-like"
- "Google AI Studio Build equivalent"
- "editable pptx ready" unless native text boxes exist in the output
- "whitepaper ready" when describing infographic mode output
- "parity" with any named product
- "equivalent" to any named product

## Benchmark Definition

The benchmark is abstract learning only:

- Single workspace
- Guided choices
- Outline-first generation
- Truthful export labeling

This is NOT a product comparison or feature-parity claim.

## Safe Phrases

- "inspired by"
- "guided choice flow similar to"
- "benchmark: low cognitive load"
- "takes cues from"
- "follows the pattern of"

## Unsafe Phrases (Do Not Use)

- "NotebookLM-like"
- "parity"
- "equivalent"
- "editable PowerPoint"
- "same as"
- "matches"
- "replaces"

## Rationale

Product comparison claims create user expectations that the system cannot fulfill. The project provides a guided infographic generation pipeline with specific export formats (PDF, ZIP, HTML). Overstating capability damages trust.
