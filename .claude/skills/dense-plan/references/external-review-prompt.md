## プロンプト

```
以下はReactアプリ（Vite + React 19 + TypeScript 5.8 + Tailwind CSS v4）の変更計画書です。
実装者が見落としている致命的な問題を指摘してください。

### 特に注意してほしい点

1. **操作フローの破壊**: 変更前後でユーザーの操作フローが壊れないか
2. **型の不整合**: TypeScriptの型エラーが発生しないか
3. **状態管理の整合性**: React useStateの更新順序、依存関係は正しいか
4. **既存機能の破壊**: 既存の動作を意図せず壊していないか
5. **具体性の不足**: 曖昧な記述、file:line参照のない変更はないか
6. **スコープクリープ**: 不要な変更が混入していないか
7. **検証基準の妥当性**: 検証手順が実際の問題を検出できるか

### フォーマット

指摘は以下のフォーマットでお願いします:
- **Critical**: 実装すると確実に壊れるもの
- **Major**: 壊れる可能性が高いもの
- **Minor**: 改善すべきだが動作には影響しないもの

---

[以下に計画書を貼り付け]

---

[以下に変更対象ファイルの現在のコードを貼り付け]
```
