# 90-Point Rubric - 2026-03-08

## Overview

This rubric defines 90-point quality achievement for infographic decks. Each deck is scored out of 100 points across 5 categories (20 points each).

## Scoring Scale

| Score Range | Quality Level | Description |
|-------------|---------------|-------------|
| 18-20 | Excellent | Ready for submission with no issues |
| 15-17 | Good | Minor polish needed, acceptable for most contexts |
| 12-14 | Fair | Visible defects, needs attention before submission |
| 0-11 | Poor | Must fix before submission |

## Categories (20 points each)

### 1. Background Cleanliness (20 points)

**18-20 (Excellent)**:
- White fallback: 0
- Background text detected: 0
- Compose warnings: 0 or minimal (1-2)
- Hybrid mode stable across all themes
- Quality gate passes
- Visual review shows clean, appropriate backgrounds

**15-17 (Good)**:
- White fallback: 0
- Background text: 0-1
- Compose warnings: 0-2
- Minor visual issues (slight noise, slightly busy)
- Still acceptable for business use

**12-14 (Fair)**:
- White fallback: 0-1
- Background text: 1-2
- Compose warnings: 2-4
- Noticeable visual distractions
- May need adjustment before sensitive sharing

**0-11 (Poor)**:
- White fallback: 1+
- Background text: 3+
- Compose warnings: 5+
- Legacy mode frequently used
- Significant visual problems

### 2. Layout Safety (20 points)

**18-20 (Excellent)**:
- No element collisions (text over text, KPI over facts, footer over content)
- Appropriate spacing between all elements (headline→subheadline, KPI→facts, content→footer)
- No overflow or clipping
- Consistent margins and safe zones respected

**15-17 (Good)**:
- No critical collisions
- Minor spacing issues (tight but not overlapping)
- Occasional minor overflow (handled gracefully)
- Mostly consistent layout

**12-14 (Fair)**:
- Some element overlap or tight spacing
- Noticeable overflow in some cases
- Inconsistent safe zone handling
- Layout feels crowded in areas

**0-11 (Poor)**:
- Frequent element collisions
- Severe overflow/clipping
- No respect for safe zones
- Layout breaks in multiple areas

### 3. Readability and Hierarchy (20 points)

**18-20 (Excellent)**:
- Clear visual hierarchy (headline > subheadline > body)
- Appropriate font sizes for all text elements
- Good contrast (text vs background)
- Balanced information density (not too dense or too sparse)
- Bullet points well-spaced and readable

**15-17 (Good)**:
- Clear hierarchy overall
- Minor font size issues in specific areas
- Acceptable contrast
- Mostly appropriate density
- Bullets readable but could be more spaced

**12-14 (Fair)**:
- Hierarchy somewhat unclear
- Font sizes inconsistent (some too small/large)
- Contrast issues in areas
- Information density too high or too low
- Bullets tight or hard to scan

**0-11 (Poor)**:
- No clear hierarchy
- Font sizes inappropriate (too small to read, too large for space)
- Poor contrast throughout
- Unreadable due to density or spacing
- Bullet structure broken

### 4. KPI/Source Integrity (20 points)

**18-20 (Excellent)**:
- KPI values accurate and clearly displayed
- Units properly formatted and positioned
- Source attribution clear and legible
- No data contract violations
- KPI hierarchy makes sense (hero > supporting)

**15-17 (Good)**:
- KPI values accurate
- Minor formatting issues with units
- Source attribution present but could be clearer
- Minor data contract issues (awkward splits)
- KPI hierarchy mostly logical

**12-14 (Fair)**:
- Some KPI display issues
- Unit formatting inconsistent
- Source attribution unclear or missing in places
- Noticeable data contract violations
- KPI hierarchy confusing

**0-11 (Poor)**:
- KPI values inaccurate or missing
- Unit formatting broken
- No source attribution
- Severe data contract violations
- KPI hierarchy completely broken

### 5. Business Presentability (20 points)

**18-20 (Excellent)**:
- Professional polish throughout
- CTA clear and actionable
- Footer/source/footer area well-organized
- Appropriate use of color and branding
- Ready for external sharing

**15-17 (Good)**:
- Mostly professional
- CTA clear but could be stronger
- Minor footer/source organization issues
- Appropriate use of design elements
- Acceptable for most business contexts

**12-14 (Fair)**:
- Some unprofessional elements
- CTA weak or unclear
- Footer/source area messy
- Design inconsistencies
- Needs polish before external sharing

**0-11 (Poor)**:
- Unprofessional appearance
- No clear CTA
- Footer/source broken
- Design broken throughout
- Not suitable for business use

## PageKind-Specific Considerations

### Cover Page
- Hero KPI presence and hierarchy
- Headline → subheadline → KPI spacing
- Visual impact of hero KPI
- Footer not interfering with hero content

### Executive Summary / Section Grid
- Facts vs KPI balance
- Information density (not too crowded)
- Footer reserve space
- Source attribution visibility

### Comparison Page
- Row height consistency
- Subheadline vs body text balance
- Footer/source area readability
- No text wrapping issues in cards

### Roadmap Page
- Phase spacing consistency
- Bullet hierarchy (phase > bullets)
- 3-column layout readability
- Footer not overlapping timeline

### Decision-CTA Page
- Actions → CTA spacing
- CTA prominence
- Source clarity
- Footer reserve
